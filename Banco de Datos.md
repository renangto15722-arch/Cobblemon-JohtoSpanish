# Traducción al Español de Johto

Este repositorio contiene la traducción al español del mapa de aventuras Johto para Cobblemon, creado por Jond.
Este proyecto únicamente reemplaza los textos del juego por su traducción al español y requiere el datapack original para funcionar.

## Proyecto original

- Pixelmon Johto:
  https://www.curseforge.com/minecraft/worlds/pixelmon-johto
- Cobblemon Johto:
  https://www.curseforge.com/minecraft/worlds/cobblemon-johto

## Créditos

Mapa y datapack originales creados por Jond.
Traducción al español por RenanBatatinha123.

## Actualizaciones

A medida que el proyecto original reciba nuevo contenido y diálogos, este repositorio se actualizará para ofrecer la traducción al español de la versión más reciente.
